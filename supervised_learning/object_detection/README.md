# Object Detection - YOLO v3

This project implements a YOLO v3 object detection system using TensorFlow and Keras. It includes classes for model initialization and output processing.

## Tasks
- **0. Initialize Yolo**: Setup the class to load models and configuration.
- **1. Process Outputs**: Transform raw model predictions into scaled boundary boxes and probabilities.

## Requirements
- Python 3.8
- TensorFlow 2.15
- NumPy 1.25.2
