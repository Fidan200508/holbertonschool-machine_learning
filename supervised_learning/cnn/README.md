# Convolutional Neural Network - Forward Propagation

## 📌 Project Overview
This project implements **forward propagation for a convolutional layer** in a Convolutional Neural Network (CNN) using NumPy.

The goal is to understand how convolution operations work internally without relying on deep learning frameworks like TensorFlow or PyTorch.

---

## 🧠 Concepts Covered
- Convolution operation
- Padding (`same` vs `valid`)
- Stride
- Multi-channel input
- Multiple filters (kernels)
- Activation functions (ReLU)
