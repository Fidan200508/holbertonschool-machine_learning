# Advanced Linear Algebra & Dimensionality Reduction

This repository contains implementations of fundamental mathematical operations used in DeepTech and AI-driven automation.

## Math Tasks

### 0. Determinant
- **File:** `0-determinant.py`
- Calculates the determinant of a square matrix using recursion.

### 1. Minor
- **File:** `1-minor.py`
- Calculates the minor matrix of a square matrix.

### 2. Cofactor
- **File:** `2-cofactor.py`
- Calculates the cofactor matrix (minor matrix with alternating signs).

### 3. Adjugate
- **File:** `3-adjugate.py`
- Calculates the adjugate matrix (transpose of the cofactor matrix).

### 4. Inverse
- **File:** `4-inverse.py`
- Calculates the inverse of a matrix using the adjugate and determinant.

### 5. Definiteness
- **File:** `5-definiteness.py`
- Determines if a matrix is Positive/Negative (semi)definite or Indefinite using NumPy eigenvalues.

## Unsupervised Learning Tasks

### 0. PCA (Principal Component Analysis)
- **File:** `0-pca.py`
- Reduces dimensionality while maintaining a specified fraction of the dataset's variance using SVD.
